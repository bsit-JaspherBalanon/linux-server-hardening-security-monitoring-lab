# Project Overview

The Linux Server Hardening and Security Monitoring Lab was developed to gain hands-on experience in Linux system administration and cybersecurity operations. The project focuses on deploying an Ubuntu Server, implementing security controls, monitoring authentication logs, and detecting suspicious activities.

The lab environment was created using Oracle VirtualBox and Ubuntu Server. Security tools such as UFW Firewall and Fail2Ban were installed and configured to improve system security. Authentication logs were monitored to observe and analyze security-related events.

This project demonstrates foundational cybersecurity skills relevant to SOC Analyst, Cybersecurity Analyst, and System Administrator roles.
