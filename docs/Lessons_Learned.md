# Lessons Learned

This project provided practical experience in Linux administration and cybersecurity operations.

Key lessons learned include:

* Linux servers require continuous monitoring.
* Firewalls are essential for protecting systems.
* Fail2Ban helps prevent brute-force attacks.
* Authentication logs provide valuable security information.
* Real-time monitoring can help identify suspicious activities quickly.
* Virtualization is useful for building cybersecurity labs safely.

The project strengthened skills in system administration, security monitoring, and incident detection.
